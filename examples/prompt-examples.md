## Case Study: Refactoring Nested If-Else Statements

### Input Prompt
```markdown
Identify and refactor all nested if-else structures in the following file. Use early return guard clauses to flatten the structure.
```

### Resulting Output
- **Before**: 4 levels of nested structures.
- **After**: Flattened functions using linear guard checks. Readable at a glance, maintaining same unit tests coverage.
