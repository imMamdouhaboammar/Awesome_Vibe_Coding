# Content Skills

Rules and guidelines for writing documentation, blog posts, and technical content.

## Skills Directory

| Skill | Use Case | Target Deliverable |
|---|---|---|
| `fact-checker` | Verifies technical API names and links in draft documentation | Markdown verification |
| `readability-editor` | Adjusts technical content to fit a target readability score (e.g., Flesch-Kincaid) | Edited document |

*To add a new skill, follow [`templates/skill-template.md`](../../templates/skill-template.md).*
