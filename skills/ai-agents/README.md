# Agent Skills

Configurations and instructions to equip autonomous AI agents with specialized capabilities.

## Skills Directory

| Skill | Use Case | Required Tools |
|---|---|---|
| `codebase-mapping` | Traverses a repository to construct a structural module dependency map | Filesystem reader, Git |
| `network-tracing` | Captures and logs API request headers and payloads during execution | Network proxy or browser console |

*To add a new skill, follow [`templates/skill-template.md`](../../templates/skill-template.md).*
