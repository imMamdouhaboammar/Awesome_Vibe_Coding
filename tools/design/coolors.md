## What it is
Coolors is a fast, web-based color palette generator that lets creators design, lock, and export cohesive color themes.

## Why it matters
Provides design systems with balanced hue selections, exportable immediately to CSS variables (HEX, RGB, HSL) to style web applications consistently.

## Best use cases
- Formulating dark/light theme palettes.
- Retrieving contrasting color pairs matching accessibility guidelines.
- Standardizing design tokens.

## How it helps
- Saves time picking color codes manually.
- Provides immediate exportable CSS property strings.
- Visualizes palette combinations on sample dashboards.

## Installation prompt
```txt
Create a CSS file named index.css containing a dark mode and light mode theme variable block.
Use the following Coolors color palette parameters to define standard naming variables:
- Primary: #8B5CF6
- Secondary: #3B82F6
- Accent: #10B981
- Background: #0B0F19
Include utility classes matching these variables.
```

## Usage prompt
```txt
Write a responsive card component in HTML using the CSS variables we defined in index.css.
Ensure the text color contrast matches WCAG AA standards against the card background variable.
```
