## What it is
Perplexity AI is a citation-driven conversational search engine that scans the web in real-time to answer complex questions and summarize sources.

## Why it matters
AI model training sets are static and age quickly. Perplexity allows developers to search for recently released framework APIs, library changes, and dependency updates with direct citations.

## Best use cases
- Investigating recent changes in API schemas (e.g. Next.js 15 routing parameters).
- Finding solutions for newly reported runtime issues.
- Quick tech-stack comparisons.

## How it helps
- Reduces search engine browsing through ad-bloated links.
- Synthesizes answers directly with multiple citations to verify facts.
- Writes code blocks reflecting the latest package versions.

## Installation prompt
```txt
I need to query the latest API interface changes.
Use Perplexity or a web search tool to find:
1. The exact syntax changes between React Router v6 and v7.
2. Provide a migration code example for router definitions.
3. List the source links.
```

## Usage prompt
```txt
Search the web for any open bugs or issues related to installing [package_name] on macOS Sequoia.
Summarize the workarounds and provide the terminal command to resolve it.
```

## Notes
Copilot or deep research queries have hourly limits based on account subscription tiers.
