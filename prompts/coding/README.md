# Coding Prompts

Prompts designed to assist in code generation, refactoring, test suite creation, and systematic debugging.

## Prompts Directory

| File | Use Case | Target Models |
|---|---|---|
| `refactor-complexity.md` | Simplifies highly nested structures and reduces cyclomatic complexity | Claude 3.5 Sonnet, GPT-4o |
| `write-unit-tests.md` | Generates exhaustive unit tests for pure functions | All models |

*To add a new prompt here, follow [`templates/prompt-template.md`](../../templates/prompt-template.md).*
